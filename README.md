# Web PHP MVC – Shoe Shop 

**Web PHP MVC – Shoe Shop** là một website bán giày được xây dựng bằng **PHP thuần theo mô hình MVC**.  
Hệ thống hỗ trợ **3 phân quyền**: **Admin – Staff – User**, đáp ứng đầy đủ các chức năng quản lý và mua sắm trực tuyến.

🔗 Repository: https://github.com/HitroxVN/webphpmvc.git

---

## 🚀 Tính năng chính

### 👤 Người dùng (User)
- Đăng ký / Đăng nhập
- Xem danh sách & chi tiết sản phẩm
- Xem hình ảnh, size, color của sản phẩm
- Thêm sản phẩm vào giỏ hàng
- Đặt hàng & thanh toán
- Xem lịch sử đơn hàng
- Quản lý thông tin cá nhân

### 🧑‍💼 Nhân viên (Staff)
- Đăng nhập hệ thống riêng
- Quản lý sản phẩm
- Quản lý đơn hàng
- Xem thông tin người dùng

### 👑 Quản trị viên (Admin)
- Dashboard thống kê
- Quản lý danh mục sản phẩm
- Quản lý sản phẩm & biến thể
- Quản lý đơn hàng
- Quản lý người dùng
- Thống kê doanh thu

---

## ⚙️ Công nghệ sử dụng

- PHP Core
- MySQL
- HTML / CSS / Bootstrap
- Apache (.htaccess – URL Rewrite)
- PHP Session

---

## 🛠️ Hướng dẫn cài đặt

### 1️⃣ Clone project
```bash
git clone https://github.com/HitroxVN/webphpmvc.git
```

---

### 2️⃣ Cài đặt môi trường

- Cài XAMPP / WAMP / Laragon

- PHP >= 7.4

- Apache + MySQL

### 3️⃣ Cấu hình database

- Tạo database trong MySQL (ví dụ: shoe_shop)

- Import file:
```bash
shoe_shop.sql
```
